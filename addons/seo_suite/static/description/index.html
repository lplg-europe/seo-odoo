SEO Suite

An open-source SEO toolkit for Odoo

Most Odoo SEO modules focus on URL rewrites, schema or breadcrumbs. SEO Suite adds on-page SEO analysis inside Odoo.

Module 1 — On-page SEO audit

  • Crawl a URL, free and without any API key.
  • Title, meta description, H1, canonical, robots.
  • Internal/external links, images (and images without alt), word count.
  • Automatic detection of on-page SEO issues.

Coming next: multi-page crawl, Google Search Console, keyword research, AI recommendations.
